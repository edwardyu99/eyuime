# 余氏中文輸入法 (eyuime)

余氏中文輸入法是一套旨在融合「廣東話拼音」、「速成碼」與「三碼倉頡」三大輸入法優點的創新中文輸入系統。本專案以實用為原則，讓使用者無須頻繁切換模式，即可同時依廣東音或倉頡/速成碼輸入中文。

## 🌟 專案特色

* **三合一免切換**：廣東碼、速成碼、三碼倉頡同時並行，用家無需切換即可混合使用。
* **高效 3+3 編碼**：雙字詞採用每字各取三碼的拼合邏輯，內建逾十萬組詞庫，有效降低重碼率至約 26%。
* **極低重碼率**：常用字重碼率低於 1%，次常用字約 3%，平均按鍵可達 2 碼出一字。
* **完整支援**：支援香港常用字、日文字母、注音符號及表情符號 (Emoji)。
* **多平台版本**：支援 Windows (IME/TSF 小狼毫)、Linux (gcin/ibus) 及 Android (同文輸入法)。

## ⌨️ 取碼規則簡介

### 1. 廣東拼音碼
沿用港式粵語拼音，並針對輸入效率進行優化：
* **兼容性**：支持常見姓氏拼法（如 Chan、Lee、Kwok）。
* **簡化規則**：通常取拼音首三字母（如 `boon` ➔ `boo`，`yuen` ➔ `yue`）。
* **港式習慣**：韻母根據發音習慣調整（如 `sam` ➔ `sum`，`sap` ➔ `sup`）。

### 2. 余氏三碼倉頡
* **規則**：取碼為「(頭、頭尾)」或「(頭尾、尾)」。
* **精確選字**：重碼字於末尾加上 `z` 變為四碼以減少選字負擔。

### 3. 標點符號與一碼快速輸入
| 類別 | 輸入鍵位 | 對應字/符號 |
| :--- | :--- | :--- |
| **余氏常用字** | i, n, s, w, y, z | 我, 不, 的, 了, 是, 之 |
| **速成常用字** | a, b, c, d, e, f, g, j, k, l, m, ... | 日, 月, 金, 木, 水, 火, 土, 十, 大, 中, 一... |
| **常用標點** | h, t, ul, uq, us, uy, uz | ， 。 ！ ？ ； ： 、|   x = 標點選列 、.。．，
| **括號與書名號** | ua/ub, uc/ud, ue/uf, ug/uh, ui/uj | 「」（）『』《》【】 |

## 🎵 核心技術：3+3 組詞編碼

本方案針對 103,312 個雙字詞進行了科學化的編碼設計：
* **聲調標示**：在組詞中引入 `e`（第 4 聲）與 `z`（第 6 聲）來區分同音字。
* **韻母簡化**：如 `ang/eng/ing` 簡化為 `ag/eg/ig`。
* **聲母優化**：`ch/sh` 多數簡化為 `c/s` 以維持三碼長度。

## 🛠️ 安裝與使用

本專案提供整合好的自動安裝包，支援 Windows 7 及以上版本 (64位元)，無需手動編輯設定檔即可立即使用。

### 下載及安裝步驟
1. **下載安裝檔**：前往 [官方網站](https://sites.google.com/site/eykmime/)，下載最新版本的 `.zip` 壓縮檔並完成解壓縮。
2. **執行安裝**：執行資料夾中的 `setupReneeRime.exe`。
3. **自動配置**：安裝程式會自動安裝小狼毫輸入法引擎 (RIME/Weasel)，並部署余氏輸入法專屬碼表、設定檔與圖示。
4. **切換使用**：於系統語言列切換至「小狼毫」，並選擇「余氏輸入法」方案。

### 舊版與其他平台支援
* **Windows IME 版**：若需下載適用於舊版 Windows 7/10/11 (32/64位元) 的 IME 版本，請至 [官方下載頁面](https://sites.google.com/site/eykmime/eywin98)。
* **其他平台**：非 Windows 64 位元用戶（如 Linux 或 Android），請參考專案內對應資料夾的配置說明進行手動安裝。

### ⚠️ 注意事項
* **系統需求**：本安裝包專為 Windows 7/10/11 之 64 位元系統設計。
* **儲存文件**：安裝過程中可能會短暫關閉執行中的程式，建議先儲存工作內容。
* **重啟建議**：安裝完畢後建議重新啟動電腦。
