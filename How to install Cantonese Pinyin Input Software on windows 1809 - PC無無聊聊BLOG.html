How to install Cantonese Pinyin Input Software on windows 1809

Leave a reply

教學 – 安裝 CPIME 廣東話拼音輸入法 – Windows 1809 可用

Download and install 耶魯（Yale）from http://cpmie.hk
I personally prefer 耶魯（Yale）, the following installation method works for 粵拼（Jyutping / LSHK）, 教院式 and 劉錫祥（Sidney Lau）input as well

 

Copy and paste the following and create a setting.xml file on C:\cantonese\

<gs:GlobalizationServices xmlns:gs="urn:longhornGlobalizationUnattend">
<gs:UserList>
<gs:User UserID="Current" CopySettingsToDefaultUserAcct="true" CopySettingsToSystemAcct="true"/></gs:UserList>
<gs:InputPreferences>
<!--https://www.youtube.com/uccoffee-->
<gs:InputLanguageID Action="add" ID="0C04:E03D0C04"/>
</gs:InputPreferences>
</gs:GlobalizationServices>
<!--please watch some of my youtube video if you copy my xml, thanks-->

Then run the following command by using command prompt

control.exe intl.cpl,,/f:”c:\cantonese\setting.xml”

之後就可以用廣東話拼音打中文了

這個方法能解決 Windows 1809 不能選CPIME 和 缺少舊版control panel 的問題
This method bypass the input greyed out in the Settings and force installation

1803 的
shell:::{BF782CC9-5A52-4A17-806C-2A894FFEEAC5}
已經不能用了

配合 CPIME使用
Ver 6.2.7 (14 July 2010) 6.14MB
粵拼（Jyutping / LSHK）
教院式
劉錫祥（Sidney Lau）
耶魯（Yale）
