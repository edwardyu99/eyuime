📘 余氏輸入法 3+3 拼音編碼說明

版本：二０二六年七月十五日

余氏詞庫雙字詞拼音是 3+3 編碼。在 88,516 雙字詞中，重碼字 7,278，重碼率 8.22%。
方案利用三碼簡化、韻母和聲調變化以減少重碼。

🔤 (一) 三碼簡化

  1. 一般情況，取頭三個字母
    例：半 boon → boo，村 cuen → cue，廣 gwong → gwo，歸 gwai → gwa，骨 gwet → gwe，葵 kwai → kwa，困 kwun → kwu
  2. ang、eng、ing、ong、ung 簡化
    改為 ag、eg、ig、og/on、ug
    例：等 dang → dag，病 beng → beg，定 ding → dig，劏 tong → tog，動 dung → dug。
    *第1聲 ong → og，其餘 ong → on。例：劏房(togfon)。 常用字：光kog
  3. uet 簡化為 ut
    例：月 yuet → yut，雪 suet → sut
  4. ch 和 sh 簡化為 c 和 s
    但以下常用字保留 ch/sh 成三碼：茶查cha，長詳che，子chi(情chg)，曹cho，廚chu，沙sha，上常she，使試市shi(成shg)，數sho，信shu。
    *第1聲 eung → eo(ng)。例：昌ceo，將jeo，香heo，相seo，相信(seoshu)
  5. ng- 聲母簡化
    藝、額、暗、眼、咬、外、岳等字的 ng- 省略
    例：藝 ngai → ai，愛 ngoi → oi
    例外：銀 ngan → un，牛偶 ngau → ngu
  6. 香港姓氏及常用字取頭三碼
    例：陳 cha(n)，李 lee，張 che(ung)，黃 won(g)，何 ho，歐 au，周 cho(w)，胡 wu，馬 ma，麥 mak，史 sze，廖 liu，林 lam，白 bak，蔡 tso(i)，鄭 che(ng)，蔣 chi(ang)，朱 chu，崔 chu(i)，鍾 chu(ng)，葉 ip，金 kam，郭國 kwo(k)，關軍君 kwa(n)，施史司 sze，鄧 tan(g)，曾 tsa(ng)，謝 tse，徐 tsu(i)。
    仔 cha(i)，chi 子，記 kee，德 tak，達 tat，青 tsi(ng)，荃 tsu(en)，慈 tsz，任 yam。
    + 本 bue(n)，氣 hee，美 mee

韻母備註

[#1] aa, a 長短音的分別： 余氏輸入法跟隨港式習慣，將長音[啊] aa 簡化為 a。短音 a 變為[矮] ai。例如：沙 sa，西 sai。
[#2] 挨(ai)變為矮(ai)： 原 ai 仍可單字與 i/ie 共用。例：拜 bai/bi。為減少重碼，余氏用 i/ie：(cjsy字不分調，例: cie猜踩柴，jie齋債寨，sie嘥徙晒)。非 cjsy 字1聲用 i，其餘2-6聲用 ie。例：bi拜 bie擺敗，di大 die歹帶，fi快，gi街 gie解介，hi嗨 hie鞋孩械，li拉 lie瀨，mi買 mie賣埋，ni奶 nie乃，pi派 pie牌排，ti太呔 tie泰，wi歪 wie淮懷壞。
[#3] am, an, ao, ap, at, ak 長短音： 通常 am→um，an→en，ao→au，ap→up，at→et。例：三sam心sum，山san身sen，交gao九gau，圾sap十sup，察cat七cet*，八bat不bet*一yet*，百bak北bek (*常用字)。殺sat/失set，灣患wan/溫運wen，因忍印yen。短音常用字跟隨港式習慣，例：民文 man 雲 wan 人仁 yan，日逸 yat。
[#4] 衣(i)音的處理： 單字 i/ee 可互用。
[#5] 柯(or/oh)： 缺聲母的字拼成 or。例：柯 or，波 bor，播 boh。
[#6] 奧(o/ou)： 缺聲母的字拼成 ou。例：奧澳 ou。
[#7] 烏(uː)、於(u)ː： 余氏合併為(u)。例：胡 wu，語 yu。
[#8] 碗(un) / 冤(un)： 碗一般用 oon，例：goon觀管貫 boon般半胖，moon們滿悶。例外：本bue 門mun 盤pun。冤一般用 uen，例：guen捐卷眷 yuen院怨遠願。例外：yoo冤淵 yun元完原員園圓源 yun潤閏。

🎶 (三) 聲調

余氏單字拼音一般不分聲調。但為了減低重碼，余氏 3+3 組詞方案用了聲調字母 a啞 h氣 e兒 u雨 z二 (2-6聲)。

調類 例字 a e i o u yu un or ci ji si yi
陰平 1分-----於奀柯痴之師衣
陰上 2粉hhieoaua瘀 yuaenoh此 ci只 jie使 shi倚 yi
陰去 3訓hhieouuuyuuenoh次 chi志 jee試 shi意 yi
陽平 4焚hhieoeuu如 yueenoh持 cee-時 see兒 yee
陽上 5憤hhieouuu雨 yuuenoh似 chi-市 shi已 yie
陽去 6份hhieozuz預 yuzenoh-自 jiz是 siz二 yiz

聲調備註

[#9] a阿, e些 (2至6聲 ah, eh)： ce車 ceh且，ga加 gah假價，ha哈 hah吓霞夏，se些 seh捨卸蛇社射。
[#10] i挨 (2至6聲用 ie)： bi拜 bie擺敗，di大獃 die歹帶，fi快，gi街 gie解介，hi嗨 hie鞋孩械，li拉 lie瀨，mi買 mie賣埋，ni奶 nie乃，pi派 pie牌排，ti太呔 tie泰，wi歪 wie淮懷壞。(cjs不分調，用 ie: cie猜踩柴，jie齋債寨，sie嘥徙晒)
[#11] o奧 (2至6聲 ou)： bo煲，bou補報步，pou葡 tou圖，lou老 (*缺聲母的[奧澳]字拼成 ou)。
[#7] u烏 / yu於 (2至6聲 uu，yu: 4聲 yue)： fu夫，fuu虎富乎婦父。su書 suu鼠恕殊樹。yu於，yuu瘀，yue如，yuu雨預。
[#12] un奀 (bfgs第1聲 un，其他 en)： bun奔 / ben品殯笨；fun分 / fen粉訓焚憤份；gun根 / gen緊近；sun新 / sen呻神慎；hen很痕恨。
[#13] or柯 (1聲 or，2至6聲 oh)： cor初，coh楚錯鋤坐，sor疏，soh鎖傻。(常用字：jor左 hor可 sor所)
[#14] yi衣 ci/ji/si/yi (2至6聲)： 為減少重碼，余氏3+3碼用聲調字母 ee, iz (代替4,6聲)。特別處理：chi/jie/jee/shi/yie。例：ci雌此 / chi次似 / cee持；ji之 / jie只 / jee志 / jiz自；si師 / shi使試市 / see時 / siz是；yi衣倚意 / yee兒 / yie已以 / yiz二。
